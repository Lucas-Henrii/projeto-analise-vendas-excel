## 📊 Projeto 1: Análise de Desempenho de Vendas

Objetivo do Projeto:
* Tratar uma base de dados bruta de vendas em formato CSV e construir um dashboard interativo para apoiar a tomada de decisão do Diretor Comercial.

Problemas de Negócio Resolvidos:
* Qualidade dos Dados: Removi registros duplicados (ID 1002), limpei espaços em branco em strings e padronizei categorias de produtos de "eletrônicos" para "Eletrônicos".
* Métrica de Faturamento: Calculei a receita total por linha e consolidei o faturamento total da empresa em R$ 14.870,00.
* Análise de Destaque: Identifiquei que a categoria de Eletrônicos é o principal motor de receita da empresa, representando R$ 11.400,00 do total.

Ferramentas Utilizadas:
* Excel Avançado (Fórmulas de tratamento, SOMA, SOMASE)
8 Tabelas Dinâmicas e Gráficos Dinâmicos
* Segmentação de Dados para filtros interativos regionais
